# ss Documentation

## Man Pages
- `man ss` — complete flag, filter syntax, and output format reference
- `man netstat` — legacy reference (deprecated; part of net-tools)
- `man 7 socket` — Linux socket internals
- `man 7 tcp` — TCP socket options and states

## Official Documentation
- iproute2 project (ss is part of iproute2): https://git.kernel.org/pub/scm/network/iproute2/iproute2.git
- iproute2 documentation: https://wiki.linuxfoundation.org/networking/iproute2
- Linux kernel socket state reference: https://www.kernel.org/doc/html/latest/networking/index.html
