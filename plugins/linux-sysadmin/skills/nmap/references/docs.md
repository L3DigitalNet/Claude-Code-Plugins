# nmap Documentation

## Man Pages
- `man nmap` — complete reference for all flags, timing templates, and output formats
- `man zenmap` — GUI frontend for nmap (if installed)

## Official Documentation
- nmap reference guide: https://nmap.org/book/man.html
- nmap book (free online): https://nmap.org/book/
- NSE documentation: https://nmap.org/book/nse.html
- NSE script list with descriptions: https://nmap.org/nsedoc/
- NSE script categories: https://nmap.org/book/nse-usage.html#nse-categories
- Output format reference: https://nmap.org/book/output.html
- Timing and performance: https://nmap.org/book/man-performance.html

## Distro Guides
- Arch Linux nmap wiki: https://wiki.archlinux.org/title/Nmap

## Useful External References

- BPF filter reference (for understanding scan filtering): https://biot.com/capstats/bpf.html
- CVE search via nmap scripts: https://svn.nmap.org/nmap/scripts/

## Legal and Ethical Use
- nmap legal issues: https://nmap.org/book/legal-issues.html
- Only scan networks and hosts you are authorized to scan. Port scanning unauthorized
  systems may be illegal in your jurisdiction regardless of intent.
