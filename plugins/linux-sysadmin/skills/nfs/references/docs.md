# NFS Documentation

## Man Pages
- `man exports` — /etc/exports format and all options
- `man nfs` — NFS client mount options
- `man mount.nfs` — NFS-specific mount command details
- `man exportfs` — managing the exports table at runtime
- `man showmount` — query active mounts and exports
- `man nfsstat` — NFS statistics and debugging
- `man idmapd.conf` — NFSv4 user/group identity mapping configuration
- `man nfsd` — NFS server daemon internals
