# journald Documentation

## Man Pages
- `man journalctl` — query and display entries from the systemd journal
- `man journald.conf` — journal service configuration file
- `man systemd-journald` — journal service overview and socket activation
- `man systemd-cat` — connect a pipeline or program's output to the journal
- `man systemd-journal-remote` — receive journal messages over the network
- `man systemd-journal-upload` — send journal messages to a remote server
- `man systemd-journal-gatewayd` — HTTP server for journal access
