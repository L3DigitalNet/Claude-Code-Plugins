# tcpdump Documentation

## Man Pages
- `man tcpdump` — complete flag reference and filter expression documentation
- `man pcap-filter` — BPF filter expression syntax reference (the filter language)
- `man pcap` — libpcap API and capture internals
- `man wireshark` — GUI pcap analysis tool
- `man tshark` — terminal Wireshark (packet analysis without GUI)

## Official Documentation
- tcpdump project: https://www.tcpdump.org/
- tcpdump man page online: https://www.tcpdump.org/manpages/tcpdump.1.html
- pcap-filter BPF syntax: https://www.tcpdump.org/manpages/pcap-filter.7.html
- libpcap project: https://www.tcpdump.org/#latest-releases

## Distro Guides
- RHEL packet capture guide: https://docs.redhat.com/en/documentation/red_hat_enterprise_linux/9/html/configuring_and_managing_networking/assembly_capturing-network-packets_configuring-and-managing-networking
- Arch Linux tcpdump wiki: https://wiki.archlinux.org/title/Network_Debugging#tcpdump

## Wireshark (GUI Analysis)
- Wireshark project: https://www.wireshark.org/

- Wireshark capture filter (BPF) reference: https://wiki.wireshark.org/CaptureFilters
- Note: Wireshark display filters and BPF capture filters use different syntax
