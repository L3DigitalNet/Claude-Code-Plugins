# fdisk / parted / gdisk Documentation

## Official

- fdisk man page (online): https://man7.org/linux/man-pages/man8/fdisk.8.html
- parted man page (online): https://man7.org/linux/man-pages/man8/parted.8.html
- GNU parted manual: https://www.gnu.org/software/parted/manual/
- gdisk / GPT fdisk homepage: https://www.rodsbooks.com/gdisk/

## Arch Wiki

- Partitioning (comprehensive guide covering all three tools): https://wiki.archlinux.org/title/Partitioning
- GPT (GUID Partition Table): https://wiki.archlinux.org/title/GPT
- Parted: https://wiki.archlinux.org/title/Parted

## Community

- Rod Smith's gdisk documentation (author of gdisk): https://www.rodsbooks.com/gdisk/walkthrough.html
- DigitalOcean: How To Partition and Format Storage Devices: https://www.digitalocean.com/community/tutorials/how-to-partition-and-format-storage-devices-in-linux

## Man Pages

- `man fdisk`
- `man gdisk`
- `man parted`
- `man partprobe` — re-read partition tables after changes
- `man resize2fs` — grow or shrink ext4 filesystems
- `man xfs_growfs` — grow XFS filesystems
