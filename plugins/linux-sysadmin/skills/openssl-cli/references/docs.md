# openssl CLI Documentation

## Man Pages

- `man openssl` — top-level overview, subcommand list
- `man openssl-x509` — certificate display and signing
- `man openssl-req` — CSR and self-signed cert generation
- `man openssl-genrsa` — RSA key generation
- `man openssl-ecparam` — EC key generation and curve parameters
- `man openssl-s_client` — TLS/SSL connection testing
- `man openssl-verify` — certificate chain verification
- `man openssl-pkcs12` — PKCS12 bundle import and export
- `man openssl-enc` — symmetric encryption
- `man openssl-dgst` — digest and HMAC operations
