## CLI Tools

- node-red-admin CLI: https://nodered.org/docs/user-guide/node-red-admin
  - `node-red-admin hash-pw` — generate bcrypt password hash for adminAuth
  - `node-red-admin list` — list deployed nodes
  - `node-red-admin export` — export flows to file
