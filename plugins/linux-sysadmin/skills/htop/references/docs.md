## Related Tools

- `man top` — always-available fallback (procps-ng); similar but less interactive
- `man ps` — non-interactive process snapshot; scriptable
- `man renice` — standalone renice without a TUI
- `man strace` — syscall tracing (htop can invoke on selected process)
