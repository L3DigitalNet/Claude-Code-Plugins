# lsblk Documentation

## Official

- lsblk man page (online): https://man7.org/linux/man-pages/man8/lsblk.8.html
- util-linux project: https://github.com/util-linux/util-linux

## Arch Wiki

- File systems overview (disk layout context): https://wiki.archlinux.org/title/File_systems
- Partitioning (lsblk usage for partition inspection): https://wiki.archlinux.org/title/Partitioning

## Man Pages

- `man lsblk`
- `man findmnt` — more complete mount point listing than lsblk
- `man blkid` — query block device attributes (alternative to `lsblk -f`)
- `man dmsetup` — inspect device mapper devices (LVM, LUKS)
