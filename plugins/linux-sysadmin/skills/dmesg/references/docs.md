# dmesg Documentation

## Official

- Man page (online): https://man7.org/linux/man-pages/man1/dmesg.1.html
- util-linux project (dmesg is part of util-linux): https://github.com/util-linux/util-linux
- util-linux releases: https://github.com/util-linux/util-linux/releases

## Kernel documentation

- Kernel ring buffer (printk): https://www.kernel.org/doc/html/latest/core-api/printk-basics.html
- Kernel log levels: https://www.kernel.org/doc/html/latest/core-api/printk-basics.html#log-levels
- `dmesg_restrict` sysctl: https://www.kernel.org/doc/html/latest/admin-guide/sysctl/kernel.html#dmesg-restrict

## Community

- Red Hat: how to use dmesg: https://www.redhat.com/sysadmin/dmesg-clear-check-messages
