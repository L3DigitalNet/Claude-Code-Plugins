## Man Pages

- `lxc(1)` — LXD CLI reference
- `lxd(1)` — LXD daemon
- `lxc-start(1)`, `lxc-stop(1)`, `lxc-ls(1)` — bare LXC tools
- Online via https://manpages.ubuntu.com/ — search "lxc" or "lxd"

## Image Servers

- **Ubuntu images**: `lxc image list ubuntu:` — images from Canonical
- **Community images**: `lxc image list images:` — Debian, Alpine, Arch, Fedora, etc. from linuxcontainers.org
- **Image server browser**: https://images.linuxcontainers.org/
