# ext4 Documentation Links

## Official Kernel Documentation

- [ext4 Wiki (kernel.org)](https://www.kernel.org/doc/html/latest/filesystems/ext4/index.html) — on-disk format, feature flags, journal internals
- [ext4 Data Structures and Algorithms](https://www.kernel.org/doc/html/latest/filesystems/ext4/globals.html) — superblock layout, block groups, extents

## Man Pages (online)

- [mkfs.ext4(8)](https://man7.org/linux/man-pages/man8/mkfs.ext4.8.html) — filesystem creation options
- [tune2fs(8)](https://man7.org/linux/man-pages/man8/tune2fs.8.html) — post-creation parameter adjustment
- [e2fsck(8)](https://man7.org/linux/man-pages/man8/e2fsck.8.html) — filesystem check and repair
- [resize2fs(8)](https://man7.org/linux/man-pages/man8/resize2fs.8.html) — grow and shrink ext2/3/4 filesystems
- [dumpe2fs(8)](https://man7.org/linux/man-pages/man8/dumpe2fs.8.html) — dump filesystem info and superblock
- [debugfs(8)](https://man7.org/linux/man-pages/man8/debugfs.8.html) — interactive low-level filesystem debugger
- [mke2fs(8)](https://man7.org/linux/man-pages/man8/mke2fs.8.html) — underlying tool mkfs.ext4 calls
- [mke2fs.conf(5)](https://man7.org/linux/man-pages/man5/mke2fs.conf.5.html) — default mkfs parameters config
