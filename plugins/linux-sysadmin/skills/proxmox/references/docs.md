## Man Pages (on the Proxmox host)

- `man qm` — QEMU/KVM VM management (create, start, stop, clone, snapshot, migrate)
- `man pct` — LXC container management
- `man vzdump` — backup and restore
- `man pvecm` — cluster manager
- `man pvesm` — storage manager
- `man pvesh` — REST API shell client
- `man pveupdate` — package update wrapper
