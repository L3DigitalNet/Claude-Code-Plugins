# Changelog

All notable changes to the qt-dev-suite plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-22

### Added

- 13 domain skills covering Qt architecture, signals/slots, layouts, model-view, threading, styling, resources, dialogs, packaging, debugging, QML, settings, and binding differences
- `qt-app-dev` agent — proactive Qt development specialist
- `qt-debugger` agent — proactive diagnostics for Qt errors and crashes
- `qt-app-reviewer` agent — proactive code quality review after writing Qt code
- `qt-ux-advisor` agent — proactive UI/UX review for widget naming, accessibility, and layout hierarchy
- `/qt-dev-suite:scaffold` command — scaffold a new Python/PySide6 project with pyproject.toml + src layout
- `/qt-dev-suite:new-component` command — generate a new widget, dialog, or window class
